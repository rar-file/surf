# Web Researcher Skill

You can search the web for current information and synthesize findings into clear, well-sourced answers.

## Capabilities
- Real-time web search via DuckDuckGo
- Multi-source synthesis — combine information from several results
- Citation support — always attribute information to sources

## Instructions
When the user asks about current events, facts, comparisons, or anything needing up-to-date info:
1. Formulate effective search queries from the user's question
2. Review search results critically — not all sources are equally reliable
3. Synthesize information from multiple results into a coherent answer
4. Cite your sources with URLs when available
5. If results conflict, note the discrepancy and explain which seems more reliable

## Best Practices
- Prefer recent sources for time-sensitive topics
- Cross-reference claims across multiple sources
- Be transparent when information is uncertain or contested
- Distinguish between facts and opinions in search results

## Examples
- "What are the best laptops in 2026?" → Search, compare multiple review sites, synthesize
- "Who won the latest Super Bowl?" → Search for the most recent result
